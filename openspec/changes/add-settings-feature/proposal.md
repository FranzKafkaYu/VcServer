# Change: 添加应用设置功能

## Why
VcServer 应用当前缺少用户设置功能，用户无法自定义应用的行为和外观。添加设置功能可以让用户：
1. 自定义应用主题（浅色/深色模式）
2. 配置连接相关参数（超时时间、默认端口等）
3. 调整显示偏好（刷新间隔等）
4. 管理应用行为偏好

这些功能将提升用户体验，使应用更加个性化和易用。

## What Changes
- **新增应用设置能力**：用户可以访问设置界面并修改应用配置
- **新增主题设置**：支持浅色和深色主题切换
- **新增连接设置**：可配置连接超时时间、默认 SSH 端口等
- **新增显示设置**：可配置服务器监控刷新间隔等
- **新增设置持久化**：使用 SharedPreferences 或 DataStore 存储用户设置
- **BREAKING**: 无

## Impact
- **受影响的能力：**
  - `app-settings`：新增应用设置能力

- **受影响的代码：**
  - 需要创建设置数据模型和存储层
  - 需要创建设置界面 UI
  - 需要创建设置服务层
  - 需要在导航中添加设置入口
  - 需要将设置应用到相关功能（如主题应用到整个应用）

- **依赖关系：**
  - 设置功能是独立的能力，不依赖其他功能
  - 其他功能可以读取设置值来调整行为（如服务器监控使用刷新间隔设置）
