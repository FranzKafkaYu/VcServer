## ADDED Requirements
### Requirement: 数据库导出功能
系统 SHALL 允许用户导出所有服务器配置信息到文件。

#### Scenario: 导出服务器配置成功
- **WHEN** 用户在设置界面点击"导出数据库"按钮
- **THEN** 系统请求文件存储权限（如需要）
- **AND THEN** 系统查询数据库中的所有服务器信息
- **AND THEN** 系统将服务器信息序列化为 JSON 格式
- **AND THEN** 系统将 JSON 文件保存到用户选择的位置
- **AND THEN** 系统显示导出成功提示

#### Scenario: 导出失败 - 权限被拒绝
- **WHEN** 用户点击"导出数据库"按钮
- **AND** 用户拒绝存储权限请求
- **THEN** 系统显示权限被拒绝的错误提示
- **AND THEN** 系统不执行导出操作

#### Scenario: 导出失败 - 没有服务器数据
- **WHEN** 用户点击"导出数据库"按钮
- **AND** 数据库中没有任何服务器记录
- **THEN** 系统显示"没有数据可导出"的提示
- **AND THEN** 系统不创建导出文件

### Requirement: 数据库导入功能
系统 SHALL 允许用户从文件导入服务器配置信息。

#### Scenario: 导入服务器配置成功
- **WHEN** 用户在设置界面点击"导入数据库"按钮
- **THEN** 系统请求文件读取权限（如需要）
- **AND THEN** 系统打开文件选择器让用户选择 JSON 文件
- **AND THEN** 系统读取并验证 JSON 文件格式
- **AND THEN** 系统解析 JSON 文件中的服务器信息
- **AND THEN** 系统将服务器信息导入到数据库
- **AND THEN** 系统显示导入成功提示

#### Scenario: 导入失败 - 文件格式错误
- **WHEN** 用户选择导入文件
- **AND** 文件格式不是有效的 JSON 或格式不匹配
- **THEN** 系统显示"文件格式错误"的提示
- **AND THEN** 系统不执行导入操作

#### Scenario: 导入失败 - 权限被拒绝
- **WHEN** 用户点击"导入数据库"按钮
- **AND** 用户拒绝文件读取权限请求
- **THEN** 系统显示权限被拒绝的错误提示
- **AND THEN** 系统不执行导入操作

#### Scenario: 导入时数据冲突
- **WHEN** 用户导入包含已存在服务器（相同主机和端口）的 JSON 文件
- **THEN** 系统提示用户选择处理方式（覆盖或跳过）
- **AND THEN** 根据用户选择，系统更新现有服务器或跳过重复项
- **AND THEN** 系统导入其他不冲突的服务器信息

### Requirement: 导出文件格式
系统 SHALL 使用 JSON 格式导出服务器配置信息，并包含必要的元数据。

#### Scenario: 导出文件格式正确
- **WHEN** 系统导出服务器配置
- **THEN** JSON 文件包含版本号字段（用于兼容性检查）
- **AND THEN** JSON 文件包含导出时间戳
- **AND THEN** JSON 文件包含服务器列表数组
- **AND THEN** 每个服务器记录包含所有必要的字段（名称、主机、端口、认证信息等）
