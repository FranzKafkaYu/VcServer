# Change: 实现 SSH 登录基础功能和数据持久化

## Why
VcServer 应用的核心功能是通过 SSH 连接到远程服务器进行管理。当前应用缺少：
1. 服务器添加和配置功能
2. SSH 连接和认证能力（密码和密钥两种方式）
3. 服务器信息的本地持久化存储

这些功能是应用的基础能力，必须首先实现才能支持后续的服务器状态监控、终端执行等高级功能。

## What Changes
- **新增服务器管理能力**：用户可以添加服务器，输入 IP/域名、SSH 端口等基本信息
- **新增 SSH 认证能力**：支持密码登录和密钥登录两种认证方式
- **新增数据持久化能力**：将服务器配置信息保存到本地数据库，支持后续的数据库迁移和导入功能
- **BREAKING**: 无

## Impact
- **受影响的能力：**
  - `server-management`：新增服务器管理能力
  - `ssh-authentication`：新增 SSH 认证能力
  - `data-persistence`：新增数据持久化能力

- **受影响的代码：**
  - 需要引入 SSH 客户端库（如 JSch 或 Apache MINA SSHD）
  - 需要引入本地数据库（如 Room 或 SQLite）
  - 需要创建服务器数据模型、Repository 层、Service 层
  - 需要创建添加服务器的 UI 界面

- **依赖关系：**
  - 后续的服务器状态监控功能将依赖 SSH 连接能力
  - 后续的终端执行功能将依赖 SSH 连接能力
  - 后续的数据库迁移功能将依赖数据持久化能力
